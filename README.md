## Weather App

A simple and responsive weather web application that displays real-time weather information using a weather API.

## Features
Search weather by city name
Displays temperature, humidity, and weather conditions
Dynamic weather updates based on API response
Responsive design for mobile and desktop devices
Lightweight and fast performance

## Tech Stack
HTML
CSS
JavaScript
Weather API (OpenWeatherMap)

## How It Works
The user enters a city name
The application sends a request to the weather API
The API returns current weather data
The UI updates with the received information
